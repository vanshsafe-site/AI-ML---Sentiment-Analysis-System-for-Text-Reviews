# 💬 Sentiment Analysis AI

A machine learning project that detects the sentiment of text as **Positive 😃, Neutral 😐, or Negative 😡** using Natural Language Processing (NLP).

## 🧠 Project Overview

This project uses a simple yet effective Machine Learning pipeline to analyze text sentiment.  
It is trained on labeled data and can classify user input in real time.

---

## ⚙️ How It Works

1. Text input is taken from the user
2. Text is converted into numerical features using **TF-IDF Vectorization**
3. A **Logistic Regression model** predicts sentiment
4. Output is displayed as:
   - Positive 🙂
   - Neutral 😐
   - Negative 😡

---

## 📊 Dataset

A small custom dataset was used for training containing example sentences labeled as:
- Positive
- Neutral
- Negative
